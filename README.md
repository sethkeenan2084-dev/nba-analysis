# nba-analysis
This project looks at NBA player statistics to understand long-term shooting performance and how basic statistical tools can help explain trends in sports data. First, the dataset is filtered to include only NBA regular season games. From there, the program finds the player who played the most regular seasons and calculates their three-point shooting percentage for each year. A linear regression model is used to see how that player’s shooting accuracy changed over time, and the average accuracy is estimated using the line of best fit. For seasons the player missed, the script estimates their performance using interpolation to fill in the gaps. The project also compares Field Goals Made (FGM) and Field Goals Attempted (FGA) by calculating their mean, variance, skewness, and kurtosis, and by running both paired and independent t-tests to better understand their relationship. Overall, the program is organized with small helper functions to keep the code clean and readable. One limitation is that it assumes trends change in a straight line over time, and interpolation can only estimate missing data rather than perfectly reflect real performance.
